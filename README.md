# Windows-anti-spying
Ce projet vise à renforcer la confidentialité et la sécurité en bloquant les tentatives de suivi des liens Microsoft. À l'aide d'un script PowerShell accompagné d'un fichier batch (.bat), cette solution modifie le fichier hosts pour rediriger l'adresse IP 127.0.0.1 (adresse locale) vers les URLs associées aux mécanismes de suivi de Microsoft.
